Project 2 | Rainbow's BFF

Visit The Website | GitHub Repo

I have always been into design and art, and one of the challenges when creating a new design is finding new color ideas. Therefore, I wanted to help designers to find new color scheme ideas with this web application.

I have inspired by the best color generator tools on the web while designing this web app. This is a Single Page Application website that creates randomly generated color palettes consist of 5 colors. The user can create an account, and save, edit or delete their palettes. Additionally, the user can gather RGB and HEX values of each color. I have used HTML5, CSS3, jQuery for the front end and Ruby on Rails for the back end structure. The random color generator that creates each color is written in JavaScript by me.

  • Ruby on Rails
  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • Bootstrap

Project 3 | Stumblr - Blog

Visit The Website | GitHub Repo

And here is Stumblr... The micro-blogging site that I have contributed its creation process. This is a team project that I and my team have created during General Assembly Boston's Web Development Immersive Bootcamp program. Stumblr is a microblogging website created by me, @rogerdunnhawaii, @MaxxKowalik, and @discocisco. The back-end was developed using combination of Express.js, Mongoose, and MongoDB. The front-end was developed using JavaScript and jQuery commands to communicate with the Stumblr. In order to create a blog or comment, users will have to create an account and then sign in. All accounts, blogs, and comments are stored within the API developed in the back-end.

  • Express.js
  • Mongoose
  • MongoDB
  • Bootstrap
  • JavaScript
  • jQuery
